# Agregar script de test en package.json si no existe

Crear **`tma/vitest.config.ts`**:

```typescript
import { defineConfig } from "vitest/config"
import react from "@vitejs/plugin-react"
import path from "path"

export default defineConfig({
  plugins: [react()],
  test: {
    environment: "jsdom",
    globals: true,
    setupFiles: [],
  },
  resolve: {
    alias: {
      "@": path.resolve(__dirname, "./src"),
    },
  },
})
```

NOTA: Si `@vitejs/plugin-react` no está en package.json (lo instala create-next-app solo para Next.js, no para Vitest), verificar que el `npm install` lo agrega como devDependency.
