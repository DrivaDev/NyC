# Phase 1 — UI Design Contract

> Visual and interaction contract for Phase 1: Foundation & Auth.
> Generated by gsd-ui-researcher. Verified by gsd-ui-checker.
>
> All values are locked unless noted as DEFAULT. Sources: CONTEXT.md decisions D-01–D-15,
> REQUIREMENTS.md UI-01/UI-02/UI-03/UI-07, CLAUDE.md (global + project-level).

---

## Design System

| Property | Value |
|----------|-------|
| Tool | shadcn + cult-ui |
| Preset | none (manual init — `npx cult-ui@latest init` then `npx shadcn@beta add @cult-ui/texture-card @cult-ui/texture-button`) |
| Component library | Radix UI (via shadcn base) + cult-ui texture variants |
| Icon library | lucide-react (bundled with shadcn) |
| Font | Poppins — Google Fonts (weights: 400, 700) |
| Animation | motion (Framer Motion v11 re-export) |
| Mode | Light only — no dark mode. Background is cream #FFF7ED (source: D-12, CONTEXT.md) |

---

## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline label padding |
| sm | 8px | Input internal padding (vertical), button icon gap |
| md | 16px | Form field spacing, card internal padding |
| lg | 24px | Card-to-card gap, section padding |
| xl | 32px | Auth form max-width internal padding, section breaks |
| 2xl | 48px | Page vertical centering offset |
| 3xl | 64px | Page-level top/bottom padding |

Exceptions:
- Touch targets (buttons, links): minimum 44px height for accessibility (DEFAULT — WCAG 2.5.5)
- Footer height: 48px fixed (2xl) — contains single line of text + link

## Typography

Font family: Poppins (Google Fonts) — applied globally via Next.js `localFont` or `next/font/google`.
All sizes source: REQUIREMENTS.md UI-02, CONTEXT.md D-13.

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Display / H1 | 28px | 700 | 1.2 | Page titles: "Iniciar sesión", "Crear cuenta", "Bienvenido, NyC" |
| Heading / H2 | 20px | 700 | 1.2 | Section titles, card headings: "Casos TMA", "Contratos TMA" |
| Body | 14px | 400 | 1.5 | Input labels, descriptive text, subheadings, card subtitles, footer text |
| Caption | 11px | 400 | 1.4 | Helper text under inputs, error messages, disabled card labels, nav nudges |

> Note: H3/subheading role absorbed into Body (14px, 400). Caption weight remapped from 300 to 400.
> Consolidated from 5 roles to 4 per design contract limit.

## Color

All values locked from REQUIREMENTS.md UI-01 and CONTEXT.md D-12.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#FFF7ED` | Page background, auth form surface, /tma landing background |
| Secondary (30%) | `#FED7AA` | Card backgrounds (TextureCard), input focus ring, nav surface |
| Accent (10%) | `#EA580C` | CTAs only (see reserved list below) |
| Heading | `#9A3412` | H1 and H2 text only |
| Text | `#1C1917` | Body text, labels, captions, footer text |
| Destructive | `#DC2626` | Not used in Phase 1 — no destructive actions (DEFAULT — Tailwind red-600) |

Accent (`#EA580C`) reserved for:
1. Primary submit button: "Iniciar sesión" (TextureButton variant=primary)
2. Primary submit button: "Crear cuenta" (TextureButton variant=primary)
3. Inline link in duplicate-email error: "iniciá sesión" → /login
4. Footer link text: "Driva Dev" → https://drivadev.com.ar

Accent is NOT used for: input borders, card backgrounds, heading text, disabled states.

Disabled state (placeholder cards on /tma):
- Background: `#FED7AA` at 40% opacity
- Text: `#1C1917` at 40% opacity
- Cursor: `not-allowed`
- No hover/focus effects

## Component Inventory

Components required for this phase:

| Component | Source | Variant / Notes |
|-----------|--------|-----------------|

Animation contract (motion):
- Auth form entrance: `initial={{ opacity: 0, y: 16 }}` → `animate={{ opacity: 1, y: 0 }}` — duration 0.35s ease-out
- /tma placeholder cards: `initial={{ opacity: 0, scale: 0.97 }}` → `animate={{ opacity: 1, scale: 1 }}` — stagger 0.1s per card
- No exit animations in Phase 1

## Page Specifications

### /login

Layout: Full-screen centered column on `#FFF7ED` background.
Max-width: 400px for the form card.

| Element | Spec |
|---------|------|
| Page title (H1) | "Iniciar sesión" — `#9A3412`, 28px, 700 |
| Email field label | "Email" — 14px, 400, `#1C1917` |
| Email input | type=email, placeholder="tu@nyc.com.ar" |
| Password field label | "Contraseña" — 14px, 400, `#1C1917` |
| Password input | type=password, placeholder="••••••••" |
| Submit CTA | TextureButton primary: "Iniciar sesión" — full width |
| Register link | "¿No tenés cuenta? Registrate" → /register — 11px, 400, `#1C1917` |

Error states (inline, below affected field — 11px caption):
- Invalid credentials: "Email o contraseña incorrectos"
- Empty field on submit: "Este campo es obligatorio"

### /register

Layout: Full-screen centered column on `#FFF7ED` background.
Max-width: 400px for the form card.

| Element | Spec |
|---------|------|
| Page title (H1) | "Crear cuenta" — `#9A3412`, 28px, 700 |
| Email field label | "Email" — 14px, 400, `#1C1917` |
| Email input | type=email, placeholder="tu@nyc.com.ar" |
| Password field label | "Contraseña" — 14px, 400, `#1C1917` |
| Password input | type=password, placeholder="Mínimo 8 caracteres" |
| Submit CTA | TextureButton primary: "Crear cuenta" — full width |
| Login link | "¿Ya tenés cuenta? Iniciá sesión" → /login — 11px, 400, `#1C1917` |

Error states (inline, below affected field — 11px caption):
- Email not in allowlist: "Este email no está autorizado" (source: D-10, AUTH-02)
- Email already registered: "Este email ya tiene cuenta, [iniciá sesión](#)" — link accent `#EA580C` → /login (source: D-02)
- Password too short: "La contraseña debe tener al menos 8 caracteres" (source: D-03)
- Empty field on submit: "Este campo es obligatorio"

Success flow: auto login → redirect /tma (no confirmation screen, no toast — source: D-04)
