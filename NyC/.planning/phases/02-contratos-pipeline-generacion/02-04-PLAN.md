# Plan 02-04: Route Handler — Full Pipeline Orchestration (Wave 2)

## Objective

Implement the POST Route Handler at `src/app/api/contracts/generate/route.ts`. This is the single server-side entry point that orchestrates all Wave 1 library functions into the complete contract generation pipeline.

Purpose: The wizard's Step 3 fetch() call lands here. The route must be secure (auth check + modelId validation), process all uploaded files in memory (never write to disk), call Gemini, fill the template, and return the binary .docx with the fields-completed summary header.

Output: Complete, tested Route Handler that passes the Wave 0 stubs in `generateRoute.test.ts`.

## Context

Architecture (from RESEARCH.md System Architecture Diagram):
1. Parse formData — extract modelId + notes + siteFiles[] + personFiles[]
2. Validate session (auth()) → 401 if no session
3. Validate modelId against MODELS map → 400 if unknown (prevents path traversal)
4. Load template via loadTemplateXml(model.filename)
5. Extract placeholders — highlight strategy (adenda) or label strategy (ac)
6. Process files — loop over all uploaded File objects, extract text/encode images
7. Call Gemini → Record<id, value>
8. Fill placeholders → modified XML
9. Generate buffer → Response with binary body

SECURITY REQUIREMENTS (from RESEARCH.md threat model):
- `auth()` check MUST be the FIRST operation after parsing request
- `modelId` MUST be validated against `getModelById()` — NEVER interpolate raw input into file path
- File MIME types must be checked before processing (skip unknown types)
- GEMINI_API_KEY MUST NOT be logged, returned, or included in any response header
- NO `fs.writeFile` calls with user content

RUNTIME: Do NOT add `export const runtime = "edge"` — pdf-parse requires Node.js. The default runtime is correct.

The route must also export `export const dynamic = "force-dynamic"` to prevent Next.js from statically caching POST route handling.

```typescript
import { NextRequest, NextResponse } from "next/server"
import { auth } from "@/auth"
import { getModelById } from "@/lib/contracts/models"
import {
  loadTemplateXml,
  extractHighlightPlaceholders,
  extractLabelPlaceholders,
} from "@/lib/contracts/extractPlaceholders"
import { callGemini, type GeminiPlaceholder } from "@/lib/contracts/geminiClient"
import {
  fillHighlightPlaceholders,
  fillLabelPlaceholders,
  generateDocxBuffer,
} from "@/lib/contracts/fillPlaceholders"
import { processUploadedFile } from "@/lib/contracts/extractDocText"

// Required for vercel.json maxDuration to take effect (also sets it explicitly in code)
export const maxDuration = 60

// Prevent Next.js from statically optimizing this POST route
export const dynamic = "force-dynamic"

export async function POST(request: NextRequest) {
  // ── Security: auth check FIRST ─────────────────────────────────────────────
  const session = await auth()
  if (!session) {
    return NextResponse.json({ error: "No autorizado" }, { status: 401 })
  }

  // ── Parse FormData ──────────────────────────────────────────────────────────
  let formData: FormData
  try {
    formData = await request.formData()
  } catch {
    return NextResponse.json({ error: "FormData inválido" }, { status: 400 })
  }

  const modelId = (formData.get("modelId") as string | null) ?? ""
  const notes = (formData.get("notes") as string | null) ?? ""

  // ── Security: validate modelId against MODELS map (prevents path traversal) ─
  const model = getModelById(modelId)
  if (!model) {
    return NextResponse.json({ error: "Modelo de contrato no válido" }, { status: 400 })
  }

  // ── Collect uploaded files ──────────────────────────────────────────────────
  const siteFiles = formData.getAll("siteFiles") as File[]
  const personFiles = formData.getAll("personFiles") as File[]
  const allFiles = [...siteFiles, ...personFiles]

  // ── Validate file count (DoS protection) ────────────────────────────────────
  if (allFiles.length > 20) {
    return NextResponse.json({ error: "Demasiados archivos adjuntos" }, { status: 400 })
  }

  // ── Process files in memory — no writes to disk (CONTR-15) ──────────────────
  const extractedTexts: string[] = []
  const imageParts: Array<{ inlineData: { mimeType: string; data: string } }> = []

  for (const file of allFiles) {
    // Validate file size (10 MB per file max)
    if (file.size > 10 * 1024 * 1024) continue

    const result = await processUploadedFile(file)
    if (!result) continue
    if (result.type === "text") {
      extractedTexts.push(result.text)
    } else {
      imageParts.push(result.part)
    }
  }

  // ── Load template (server filesystem — public/templates/) ───────────────────
  let zip: import("pizzip").default
  let xml: string
  try {
    const loaded = loadTemplateXml(model.filename)
    zip = loaded.zip
    xml = loaded.xml
  } catch {
    return NextResponse.json({ error: "Plantilla no encontrada" }, { status: 500 })
  }

  // ── Extract placeholders (strategy depends on model.type) ───────────────────
  let placeholders: GeminiPlaceholder[]
  if (model.type === "adenda") {
    placeholders = extractHighlightPlaceholders(xml)
  } else {
    // model.type === "ac" — label-based strategy (D-09)
    placeholders = extractLabelPlaceholders(xml).map(lp => ({
      id: lp.id,
      context: `${lp.label}: ${lp.context}`,
    }))
  }

  const totalCount = placeholders.length

  // ── Call Gemini ──────────────────────────────────────────────────────────────
  let geminiValues: Record<string, string>
  try {
    geminiValues = await callGemini(placeholders, extractedTexts, imageParts, notes)
  } catch (err: unknown) {
    const message = err instanceof Error ? err.message : "Error desconocido"
    // Do NOT log or return GEMINI_API_KEY or sensitive data
    return NextResponse.json(
      { error: `Error al procesar con Gemini: ${message}` },
      { status: 500 }
    )
  }

  // ── Fill placeholders ────────────────────────────────────────────────────────
  let modifiedXml: string
  if (model.type === "adenda") {
    modifiedXml = fillHighlightPlaceholders(xml, geminiValues, placeholders as import("@/lib/contracts/extractPlaceholders").Placeholder[])
  } else {
    modifiedXml = fillLabelPlaceholders(
      xml,
      geminiValues,
      extractLabelPlaceholders(xml) // re-extract for fill (with original label data)
    )
  }

  // ── Count completed fields ───────────────────────────────────────────────────
  const completedCount = placeholders.filter(
    p => geminiValues[p.id] && geminiValues[p.id].trim() !== ""
  ).length

  // ── Generate buffer ──────────────────────────────────────────────────────────
  let docxBuffer: Buffer
  try {
    docxBuffer = generateDocxBuffer(zip, modifiedXml)
  } catch {
    return NextResponse.json({ error: "Error al generar el documento" }, { status: 500 })
  }

  // ── Return binary response (CONTR-13) ────────────────────────────────────────
  // File bytes are now in docxBuffer — all original uploaded file bytes are GC'd.
  // No fs.writeFile calls were made (CONTR-15).
  return new Response(docxBuffer, {
    status: 200,
    headers: {
      "Content-Type":
        "application/vnd.openxmlformats-officedocument.wordprocessingml.document",
      "Content-Disposition": `attachment; filename="contrato.docx"`,
      "X-Fields-Completed": `${completedCount}/${totalCount}`,
    },
  })
}
```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client → POST /api/contracts/generate | Unauthenticated POST body with user-controlled modelId and file uploads crosses here |
| Route Handler → Gemini API | Internal server-to-server call; API key must not leak to client side |
| Route Handler → /public/templates filesystem | Server reads template files; path must be computed, not user-controlled |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Elevation of Privilege | POST /api/contracts/generate | mitigate | `auth()` called as first operation; return 401 if session is null |
| T-02-02 | Tampering | modelId parameter → file path | mitigate | `getModelById(modelId)` validates against MODELS constant map; raw input never interpolated into `path.join` |
| T-02-03 | Tampering | Malicious .docx upload | accept | mammoth.extractRawText extracts plain text only; file content never executed or inserted into template XML |
| T-02-04 | Denial of Service | File upload size / count | mitigate | Validate file count ≤ 20 and size ≤ 10 MB per file before processing loop |
| T-02-05 | Information Disclosure | GEMINI_API_KEY | mitigate | Key only in `process.env.GEMINI_API_KEY`; Gemini errors caught and sanitized before returning to client |
